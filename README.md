# Netflix Movies & Shows Analysis Dashboard

A comprehensive Power BI dashboard for analyzing Netflix's content library, viewer engagement, and discovering hidden gems among thousands of titles.

## ğŸ“Š Project Overview

This interactive dashboard provides deep insights into Netflix's vast library of movies and TV shows, combining data from multiple sources to help users discover quality content and understand viewing trends. The dashboard enables data-driven decision-making through real-time updates, drill-down capabilities, and intuitive visualizations.

## âœ¨ Key Features

- **Interactive Visualizations**: Bar charts, scatter plots, tables, and pie charts for comprehensive data exploration
- **Hidden Gem Score**: Proprietary metric identifying high-quality content with lower review counts
- **Multi-source Integration**: Combines data from Netflix, IMDb, Rotten Tomatoes, box office figures, and YouTube trailers
- **Advanced Filtering**: Search and filter by actors, directors, genre, country, release year, and awards
- **Drill-through Capabilities**: Deep-dive into specific data points with tooltips and detailed views
- **Real-time Updates**: Dashboard refreshes monthly with the latest Netflix content
- **Key Performance Indicators**: Viewer engagement, retention rates, and content popularity metrics

### Prerequisites
- Power BI Desktop (latest version)
- Access to Netflix dataset
- API keys for IMDb, Rotten Tomatoes (if fetching live data)

1. Clone this repository

1. Open the dashboard in Power BI Desktop or Service
2. Use slicers to filter by genre, year, country, or rating
3. Click on visualizations to drill-through for detailed information
4. Export reports or share via Power BI Service
